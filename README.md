# Landing CamPù — Deploy su Vercel

Questa è una landing **statica** (solo HTML + assets).

## Deploy (consigliato) — Vercel Dashboard

1. Vai su Vercel e fai **Add New → Project**
2. Importa il progetto (puoi anche fare “Upload”/import locale)
3. Se richiesto:
   - **Framework Preset**: *Other*
   - **Build Command**: lascia vuoto
   - **Output Directory**: lascia vuoto
4. Deploy

La home è `index.html`.
